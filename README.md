Para Crecer Location
==================

## Descripción

<p align="justify">
	El presente sistema pretende mostrar la ubicación en tiempo real de los usuarios.
</p>

<p align="justify">
	Para realizar el siguiente sistema tendremos dos opciones de stack según mi experiencia personal:
</p>

<b>Relacional</b>

<ol>
	<li>
		<p align="justify">
			PHP
		</p>
	</li>
	<li>
		<p align="justify">
			SQL
		</p>
	</li>
	<li>
		<p align="justify">

		</p>

<b>No Relacional</b>

<p align="justify">
